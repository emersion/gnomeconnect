# GNOMEConnect

A GNOME frontend for KDEConnect

## Installing

```bash
go get -u github.com/emersion/gnomeconnect
cd $GOPATH/src/github.com/emersion/gnomeconnect
make install-user
```

## Building

```bash
go get -u github.com/emersion/gnomeconnect
cd $GOPATH/src/github.com/emersion/gnomeconnect
make
```
